## 📋 Sobre o Projeto

Bem-vindo ao meu portfólio pessoal! Este é um site moderno e responsivo desenvolvido para apresentar meus projetos, habilidades e experiências profissionais. O portfólio foi construído com as mais recentes tecnologias web para garantir uma experiência rápida, fluida e acessível.

## ✨ Características

- ⚡ **Performance otimizada** com Next.js 16
- 🎨 **Design moderno** com Tailwind CSS 4
- 📱 **Totalmente responsivo** para todos os dispositivos
- 🌙 **Suporte a tema escuro/claro**
- 🔒 **Type-safe** com TypeScript
- 🚀 **SEO otimizado** com metadados do Next.js
- ⚙️ **Configuração ESLint** para qualidade de código

## 🛠️ Tecnologias Utilizadas

- **[Next.js 16.0.3](https://nextjs.org/)** - Framework React para produção
- **[React 19.2.0](https://react.dev/)** - Biblioteca JavaScript para interfaces
- **[TypeScript 5](https://www.typescriptlang.org/)** - Superset JavaScript com tipagem estática
- **[Tailwind CSS 4](https://tailwindcss.com/)** - Framework CSS utility-first
- **[ESLint](https://eslint.org/)** - Linter para JavaScript/TypeScript

## 📦 Pré-requisitos

Antes de começar, certifique-se de ter instalado:

- [Node.js](https://nodejs.org/) (versão 18 ou superior)
- [npm](https://www.npmjs.com/) ou [yarn](https://yarnpkg.com/)

## 🚀 Como Executar

### 1. Clone o repositório

```bash
git clone <url-do-repositório>
cd novo-portfolio
```

### 2. Instale as dependências

```bash
npm install
```

### 3. Execute o servidor de desenvolvimento

```bash
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.

## 📜 Scripts Disponíveis

No diretório do projeto, você pode executar:

- `npm run dev` - Inicia o servidor de desenvolvimento
- `npm run build` - Cria uma build de produção
- `npm run start` - Inicia o servidor de produção (após o build)
- `npm run lint` - Executa o ESLint para verificar o código

## 📁 Estrutura do Projeto

```
novo-portfolio/

```
